# Cookiecutter Vagrant project template

A simple project application scaffold with Vagrant.

Provides:

  - Vagrant multi-machine environment.
  - Very basic machine definition in `yml` file.
  - Simple Vagrant provision using shell scripts.

### Usage

Install Cookiecutter:

    $ pip install cookiecutter

Clone template:

    $ cookiecutter https://github.com/raulmartinezm/cookiecutter-vagrant.git [-o path/to/destination/]
